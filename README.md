# PureScript Neon example

This is a simple example app using [Neon](https://neon-bindings.com/) with [Purescript](http://www.purescript.org/).

## Build Setup

```npm
# install dependencies (including rust bindings)
$ npm i

# build purescript 
$ npm run build

# build for release
$ npm run release

# run
$ node ./dist/index.js
```
